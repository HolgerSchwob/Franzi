# SpaceMathe – Franziskas Mathe im Weltraum

Ein browserbasiertes Lernspiel für **Einmaleins (1×1)**, **Addition über 10** und **Subtraktion über 10**. Läuft in einer einzigen HTML-Datei, ohne Server – ideal für iPad und Desktop.

## So startest du das Spiel

- **Am PC:** `index.html` im Browser öffnen (Doppelklick oder „Öffnen mit“ → Chrome/Safari/Edge).
- **Auf dem iPad:** Siehe unten.

## Spiel auf dem iPad nutzen

### Option A: Über das Internet (empfohlen)

1. Öffne **Safari** auf dem iPad.
2. Gehe zu der Adresse, die dir dein Papa/Mama gibt (z.B. von GitHub Pages oder Netlify).
3. Tippe auf **Teilen** (Quadrat mit Pfeil nach oben).
4. Wähle **„Zum Home-Bildschirm“**.
5. Fertig – das Symbol erscheint wie eine App. Beim Tippen startet SpaceMathe ohne Browser-Leiste.

### Option B: Nur auf dem iPad (ohne Internet)

Die Datei `index.html` per AirDrop, Mail oder iCloud auf das iPad schicken. In der App **Dateien** die Datei antippen und „Öffnen mit Safari“ wählen. Danach kannst du ebenfalls „Teilen → Zum Home-Bildschirm“ nutzen.

## Was du einstellen kannst

- **Was üben?** Einmaleins, Addition über 10, Subtraktion über 10 (einzeln oder kombiniert).
- **Reihen:** Bei Einmaleins die gewünschten Reihen (2er, 3er, … 10er) ankreuzen.
- **Geschwindigkeit, Anzahl Antworten, Steigerung** wie gewohnt.

Viel Spaß im Weltraum.
